## Instalação

    - Clonar repositorio
    $ git clone https://github.com/jveiiga/project-eletrica-jsantos.git

    FRONT-END 📺
    
    $ cd project-eletrica-jsantos

    - Instalação de dependências

    $ yarn install
    ou
    $ npm install

    - Rodar projeto
    $ yarn dev
    ou
    $ npm run dev
